# 🔐 CipherLab

Interactive cryptography playground inspired by the Caesar Cipher.

Explore encryption, decryption, visual rotation, and basic cryptanalysis through an editorial-style interface.

---

## Features

Caesar Cipher Encryption  
Live Cipher Wheel Visualization  
Instant Decryption  
Copy Output  
Shift Presets  
Brute Force Attack (26 shifts)  
Plaintext Detection  
Responsive UI  
Editorial Design System

---

## Preview

Type text → Choose shift → Generate ciphertext → Break the cipher.
